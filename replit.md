# Discord Bot

Bot de Discord en Python con sistemas de bienvenida, auto-moderación y tickets profesionales.

## Run & Operate

- El bot corre automáticamente con el workflow **Discord Bot** (`python3 discord-bot/main.py`)
- Required env/secret: `DISCORD_BOT_TOKEN` — Token del bot en Discord Developer Portal

## Stack

- pnpm workspaces, Node.js 24, TypeScript 5.9
- API: Express 5
- DB: PostgreSQL + Drizzle ORM
- Validation: Zod (`zod/v4`), `drizzle-zod`
- API codegen: Orval (from OpenAPI spec)
- Build: esbuild (CJS bundle)
